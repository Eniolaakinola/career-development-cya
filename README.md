# Career Development CYA — Internship Applications

## Career Goals

**Short-term:** Land a software engineering internship for Summer 2026
to gain real industry experience before graduation.

**Long-term:** Work as a software engineer in systems or infrastructure,
and eventually build something of my own.

## Chosen Adventure

I applied to 7 software engineering internships targeting systems,
embedded, infrastructure, and enterprise software roles — all areas
aligned with my coursework in C/C++, C#, operating systems, and
computer architecture at Washington State University.

I chose this adventure because the application process itself is a
skill: researching companies, tailoring materials, and articulating
your value clearly under competitive conditions. Each application
required deliberate effort — no bulk-applying.
